# Contributing to Awesome GCP LLM Projects

First off, thank you for considering contributing! Your help is what makes this community project awesome.

## How Can I Contribute?

### Suggesting a New Project
The easiest way to contribute is to open an issue to suggest a new project. Please include a link to the project and a brief description of why it's a good fit for this list.

### Adding a Project via Pull Request
If you're comfortable with it, you can also add a project directly via a Pull Request.

1.  **Fork the repository.**
2.  **Create a new branch:** `git checkout -b add-my-awesome-project`
3.  **Add your project:** Edit the `README.md` file, adding the new project under the most relevant category. Please maintain the existing format:
    `* [Project Name](https://github.com/user/repo) - A brief, one-sentence description of the project.`
4.  **Commit your changes:** `git commit -m 'feat: Add Project Name'`
5.  **Push to your branch:** `git push origin add-my-awesome-project`
6.  **Open a Pull Request.**

## What Makes a Project "Awesome"?
This list focuses on projects directly related to building with Large Language Models on Google Cloud Platform. This includes, but is not limited to:
- Projects using Gemini, Gemma, or other models via Vertex AI.
- Implementations of RAG, agentic systems, or other patterns on GCP.
- Tools that simplify working with LLMs in a Google Cloud environment.

Thank you for your contributions!
